- Format: Full sections (Shipped, In Progress, Blockers, Bugs, Learnings, Metrics, Next 2 Weeks)
- Scope: All six boards (Identity, Integrity, Explore, DevEx, Quality Engg, Comms)

Run the Jira workflow and output the detailed template. Include JPD only when user asks.
