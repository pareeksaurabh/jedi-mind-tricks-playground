Use the **pm-updates** subagent to generate a holistic Platforms update. Default scope: all six boards (Identity, Integrity, Explore, DevEx, Quality Engg, Comms).

Ask the user: **Executive-focused or detailed (broader teams)?** Then run the full Jira workflow and produce the update. Include JPD only when user asks.
